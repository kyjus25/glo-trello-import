Trello Credentials

You will need to get your API Key and Token from Trello.

Glo Credentials

You will need to get your Personal Access Token from GitKraken.

Details

Summary

This application will take an existing Trello board and import it into GitKraken Glo! Note that Glo is under active development, and therefore not all parts of a Trello board may be copied 1:1.

One-Time Sync

This application is only a one-time import. This will not sync your items back and forth between Trello and Glo.

Security

We ask for your credentials for both services so that we can make API calls on your behalf, automating the copy process. This application is entirely client side, meaning the only time the credentials leave this page is when they are used to make those API calls.

Copied Items

  • Board Name
  • Board Columns
  • Labels
  • Card Name
  • Card Description
  • Card Tasks
